## Project Description
This project is a user management application developed in Java. The application allows 
for the registration of new users by rigorously validating their data (username, complex 
password, and international format phone number) and subsequently verifying their 
credentials during a login phase.

The project puts Object-Oriented Programming (OOP) concepts into practice by clearly 
separating responsibilities between validation logic, data storage, and the user interface.
